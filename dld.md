  │   ├── PromptStep.tsx          # 第2步: 题目                                                                                                            
  │   ├── ExecutionStep.tsx       # 第3步: 执行可视化                                                                                                      
  │   ├── ResultStep.tsx          # 第4步: 结果

  │   ├── ModelNode.tsx           # 模型节点组件                                                                                                           
  │   ├── DetailDrawer.tsx        # 详情抽屉                                                                                                               
  │   └── StatisticsCard.tsx      # 统计卡片                                                                                                               
  └── hooks/                                                                                                                                               
      └── useBattleFlow.ts        # 流程状态管理                                                                                                           
                                                                                                                                                           
  2. 可视化库选型                                                                                                                                          
                                                                                                                                                           
  推荐方案: React Flow + Framer Motion                                                                                                                     
  - React Flow: 节点图绘制、连线、交互                                                                                                                     
  - Framer Motion: 动画效果、状态转换                                                                                                                      
  - Recharts: 统计图表                                                                                                                                     
                                                                                                                                                           
  备选方案: 纯自定义SVG + CSS动画                                                                                                                          
  - 更轻量、完全可控                                                                                                                                       
  - 适合这种结构固定的流程图                                                                                                                               
                                                                                                                                                           
  3. 状态动画设计                                                                                                                                          
                                                                                                                                                           
  节点状态:                                                                                                                                                
    pending:  灰色边框 + 虚线                                                                                                                              
    running:  蓝色边框 + 脉冲动画                                                                                                                          
    success:  绿色边框 + ✅ 图标 + 放大弹跳                                                                                                                
    error:    红色边框 + ❌ 图标 + 抖动                                                                                                                    
    judging:  黄色边框 + 旋转加载                                                                                                                          
                                                                                                                                                           
  连线动画:                                                                                                                                                
    数据流动: 虚线流动动画 (dash offset)                                                                                                                   
    完成状态: 实线 + 渐变色
